# PureSleep

Independent Amerisleep mattress reviews, rankings, and buying guides. Chiropractor-reviewed. Updated monthly.

## Tech Stack
- Astro SSG (static HTML on first byte)
- React components (islands architecture)
- Tailwind CSS
- Framer Motion

## Run Locally
Prerequisites: Node.js 18+

1. npm install
2. npm run dev

## Build
npm run build
